# セクション3：第1次AIブームと冬の時代 — 学習メモ

> **テーマ：** 過剰な期待がどのように幻滅に変わるか、その構造を理解する  
> **メタ認知の層：** 第4層（認知バイアスの監視）— 自分の中にも同じ楽観がないかを点検する

## このセクションで学ぶこと

AIの最初の成功と挫折の歴史を通じて、「期待→失望→冬」のサイクルの構造を理解する。このパターンは70年後の現在も再現されており、AI導入の失敗を予測するための最初の診断ツールを手に入れる。

| 人物 | 貢献 | 覚えておくべきこと |
|------|------|-------------------|
| ハーバート・サイモン（1916–2001） | GPS（General Problem Solver）開発、4つの予言 | 「10年でチェス王者に」→実際は40年。誤差4倍の楽観が「冬」の原因の一つ |
| ジョセフ・ワイゼンバウム（1923–2008） | ELIZA開発（1966年） | 自分の秘書がELIZAに「理解」を投影した事実に衝撃を受け、AI批判者に転じた |
| フランク・ローゼンブラット（1928–1971） | パーセプトロン発明（1957年） | 「データから学ぶ機械」の最初の実現。43歳でボート事故死 |
| マービン・ミンスキー＆シーモア・パパート | 『Perceptrons』出版（1969年） | XOR問題を指摘しパーセプトロン研究を凍結。しかし多層ネットワークの学習法がないという指摘は技術的に正しかった |
| サー・ジェイムズ・ライトヒル（1924–1998） | ライトヒルレポート（1973年） | 英国政府にAI研究の打ち切りを勧告。「冬」の直接の引き金 |

## 核心概念

### ELIZAエフェクト
- 極めて単純なプログラム（パターンマッチング）に対して、人間が「理解してくれている」と感じてしまう現象
- ワイゼンバウムの秘書は、ELIZAがプログラムだと知っていても、「部屋を出てほしい」と言って二人きりで「相談」した
- **2020年代のChatGPTに対する反応と構造が同じ。ただし影響の規模が桁違い**

### 組み合わせ爆発
- 問題の規模が少し大きくなるだけで、必要な計算量が指数関数的に増大する現象
- ライトヒルが指摘したAI研究の根本的限界
- 「おもちゃの問題」では解けても、現実の問題では計算が追いつかない

### AIの冬の5段階構造
1. 過剰な期待が膨らむ
2. 約束と成果の差が明らかになる
3. メディアと世論が批判に転じる
4. 資金が削減される
5. 人材が流出し、研究が停滞する
